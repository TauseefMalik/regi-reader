# regi-reader
Registry reader for windows

# Usage
```
regreader.exe <<registry-file>> <<key>> <<value>>
```
# Usage example
```
regreader.exe C:\Users\win_user\Desktop\regdata\SOFTWARE "SOFTWARE\Microsoft\NET Framework Setup\NDP\v4\full" "Release"
```
